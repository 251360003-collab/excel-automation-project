# Excel Automation Project

This project automatically generates an Excel sales report using Python and Pandas.

## Features

- Creates Excel reports automatically
- Calculates total sales
- Exports data to .xlsx format

## Technologies Used

- Python
- Pandas
- OpenPyXL

## How to Run

```bash
pip install pandas openpyxl
python excel_report.py
```

## Output

The script generates:

```text
sales_report.xlsx
```

containing sales data and total sales.
